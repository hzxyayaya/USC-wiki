# 新生入学指南 🎒

欢迎来到 USC！作为一名新生，你可能对即将到来的大学生活充满好奇和困惑。这里为你准备了一份详细的入学清单。

## ✅ 入学前准备

### 证件类
- [x] 录取通知书
- [x] 身份证（及复印件）
- [ ] 团员证/党员档案
- [ ] 户口迁移证（如需）

### 生活用品
- [ ] 床上用品（学校通常有卖，但自带更舒服）
- [ ] 换洗衣物（查好当地天气）
- [ ] 个人洗漱用品

## 🗺️ 报到流程

1. **接站**: 学校在火车站/机场设有新生接待点。
2. **报到**: 前往各学院迎新点办理入学手续。
3. **宿舍**: 领取钥匙，入住寝室。
4. **体检**: 按安排进行入学体检。

## 📱 必备 APP

- **企业微信**: 学校官方通知发布平台。
- **教务处**: 选课、查成绩。
- **校园卡**: 充值饭卡、洗澡水费。
