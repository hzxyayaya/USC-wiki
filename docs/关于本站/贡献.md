# 有GitHub账号

## 修改
点击文章下面的`在GitHub上编辑`按钮进入GitHub修改
![[贡献-有GitHub账号-修改.png]]
如果是第一次修改需要fork一下这个仓库
![[贡献-有GitHub账号-修改-7.png]]
在这里可以看到这里提示你提交更改之后会写入到自己的分支里面，然后可以根据这个提交pull request，下面是更改的内容，更改完之后点击提交
![[贡献-有GitHub账号-修改-1.png]]
然后写好提交的内容
![[贡献-有GitHub账号-修改-2.png]]

在这里可以看到修改的内容，以及创建pull request
![[贡献-有GitHub账号-修改-3.png]]

在这里写完修改的内容之后就可以创建pull request了
![[贡献-有GitHub账号-修改-4.png]]

然后就可以在pull request当中看到提交信息了
![[贡献-有GitHub账号-修改-5.png]]

通过审核，然后merge之后就可以在网页里面看到更新的内容

## 提交issue
如果发现网页有一些bug或者想提一些建议可以在GitHub中提交issue

进入issues界面，点击New issue
![[贡献-有GitHub账号-提交issue-4.png]]

然后可以在这里提一些建议和bug
![[贡献-有GitHub账号-提交issue-1.png]]

创建完之后就可以看到issue的内容了，与这个问题有关的内容可以在这个issue下面评论
![[贡献-有GitHub账号-提交issue-2.png]]
