# 贡献指南

感谢参与 USC Wiki 共建。完整说明（目录约定、Markdown 语法、PR 流程等）见 GitHub 仓库中的 [**CONTRIBUTING.md**](https://github.com/hzxyayaya/USC-wiki/blob/main/CONTRIBUTING.md)。

## 快速开始

```bash
pnpm install
pnpm dev
```

浏览器打开 `http://localhost:4321` 预览修改。提交前请运行 `pnpm build` 确认构建通过。

Markdown 语法示例见 [[markdown-demo|本站示例页]]。
