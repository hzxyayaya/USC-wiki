## 启动校园网

启动一般有两种方式：

1. **学校统一办理** - 学校统一在班级群里面宣传激活校园卡，然后统一缴费
2. **校外商家服务** - 会有校外的商家提供校园网的激活服务，一般会在开学前一个月在宿舍门口宣传，或者直接来寝室宣传，价格一般在 100-200 元不等

## 连接校园网

### 方式一：无线网络（USC WiFi）

1. 在网络设置中搜索并连接 `USC WiFi`
2. 首次连接会自动跳转到认证页面
3. 输入账号（学号）和密码（默认为身份证后六位）
4. 认证成功后即可使用

### 方式二：有线网络

在寝室里面，可以申请连接学校的有线网。

::: tip 注意
你的电脑需要支持网线接口（RJ45），或者准备一个 USB 转网口转换器。
:::
