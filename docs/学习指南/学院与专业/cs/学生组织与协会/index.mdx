## 怎么选择

| 如果你希望…… | 可以关注 |
| --- | --- |
| 系统学习算法、组队参加竞赛 | 程序设计与算法类协会 |
| 动手做网站、应用或开源项目 | 软件开发与开源类协会 |
| 学习攻防、参加安全竞赛 | 网络安全类协会 |
| 接触机器学习、模型与数据分析 | 人工智能类协会 |

选择协会时，不必只看获奖情况。更建议确认协会的活动频率、是否提供新人培训，以及自己每周能够投入的时间。

## 提交协会资料

协会负责人可以按照统一模板提供以下材料：

- 协会名称、Logo 和一句话介绍
- 主要方向、日常活动和预计时间投入
- 是否接受零基础成员，以及新人培养方式
- 2～4 个代表活动、项目或比赛成果
- 3～6 张已获得公开展示许可的活动照片
- 本学期招新时间、公开群聊或官方账号
- 内容联系人与信息更新时间

可直接复制仓库中的 `docs/_templates/协会介绍模板.md` 填写。涉及个人手机号、私人微信等信息时，请先征得本人同意；页面优先展示公开招新群或协会官方账号。

> 协会信息建议每学期开学确认一次。超过一年未确认的内容，应标注“信息可能已过期”。
