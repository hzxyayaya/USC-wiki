# USC Wiki 首页「知性编辑」改版设计

## 目标

在不改变 Astro、Starlight 与 Galaxy 主题技术栈的前提下，重做首页的信息层级和视觉表达，让首页同时具备：

- 明确的南华学生社区品牌感；
- 更快的搜索与分类入口；
- 与现有文档页一致的蓝紫色系和明暗主题；
- 适合长期维护的克制、可信、编辑式视觉。

本次只改首页，不重构文档页、侧边栏生成逻辑或搜索实现。

## 视觉方向

采用已确认的「知性编辑」方向：将首页表现为一本持续更新的校园学生手册，而不是常见的渐变 Hero 加卡片网格。

核心视觉元素：

- 细分隔线、章节编号、刊物页码与中英文小标题；
- 中文标题使用具有书刊感的衬线字体栈，正文和交互保持现有无衬线字体；
- 大面积留白与严格网格，避免厚重阴影、玻璃拟态和大面积彩色卡片；
- 分类色只用于章节短线、序号或悬停反馈，不作为大面积背景；
- 动效仅用于首屏淡入、章节悬停和焦点反馈，并尊重 `prefers-reduced-motion`。

## 配色

颜色直接复用 Galaxy / Starlight 现有变量，确保首页和文档页属于同一套站点：

- 浅色背景：`var(--sl-color-bg)`，当前对应 `#fbfcfc`；
- 深色背景：`var(--sl-color-bg)`，当前对应 `#17181c`；
- 导航深色：沿用 Galaxy 的 `#0d1117`；
- 主强调色：`var(--sl-color-accent)`；
- 高亮文字与细节：`var(--sl-color-accent-high)`，深色模式当前为 `#b6c4fb`；
- 正文、次要文字、分隔线分别使用 `--sl-color-white`、`--sl-color-gray-*` 和 `--sl-color-hairline`。

现有分类色继续使用：

- 新生入门：`#79a7ff`
- 学习指南：`#5ec8d8`
- 校园生活：`#7bd88f`
- 事务办理：`#f3be62`
- 竞赛与资源：`#f0a060`

这些颜色只作为章节标记，不改变整体蓝紫主色。

## 页面结构

### 1. 手册抬头

首页内容顶部展示紧凑的刊物式抬头：

- 左侧：`USC WIKI / 南华手册`
- 右侧：全部目录、最近更新、参与共建等轻量入口

站点原有 Starlight 顶栏继续保留，抬头属于首页正文的一部分，用于建立页面主题，不替代全站导航。

### 2. 主封面区

三栏编辑网格：

1. 章节号 `01` 与 `THE STUDENT FIELD GUIDE`；
2. 主标题「在南华，找到你需要的答案。」和简短说明；
3. 可点击的首页搜索入口。

搜索入口调用现有 `Search.astro` 中的 Pagefind 模态框，不创建第二套搜索组件。键盘快捷键继续使用 `Ctrl/⌘ + K`。若 Pagefind 索引未生成，沿用现有搜索组件的提示信息。

### 3. 五章内容索引

将现有五个分类改为横向章节索引：

1. 初来南华：新生入门；
2. 认真学习：学习指南；
3. 好好生活：校园生活；
4. 办好事情：事务办理；
5. 走得更远：竞赛与资源。

每章包含章节号、分类标题、简短说明和分类色标记。整块可点击，并提供清晰的悬停与键盘焦点状态。

### 4. 常用入口与站点信息

章节索引下方保留现有高频入口：

- 选课指南；
- GPA；
- 校园生活指南；
- 软件推荐；
- 模板下载。

同时展示动态 `noteCount`、学生社区贡献说明和非官方免责声明。经验分享、关于本站与加入社区通过这一层和抬头导航进入，不混入五个主要章节。

## 数据与组件

- 保留 `getCollection('docs')` 计算笔记数量；
- 保留首页内的 `sections` 与 `highlights` 数据数组；
- 不引入新依赖，不改变内容集合；
- 首次实现集中在 `src/pages/index.astro`，避免为单页样式过早拆分组件；
- 若后续多个页面复用章节索引，再提取独立组件。

## 响应式设计

- 桌面端：主封面三栏，章节索引五列；
- 平板端：主封面两栏，搜索入口换到下一行；章节索引两到三列；
- 手机端：单栏阅读流，章节号缩小，搜索入口全宽，章节索引纵向排列；
- 保证中文标题不发生孤立单字换行；
- 不依赖固定高度，内容放大到 200% 时仍可阅读和操作。

## 可访问性

- 首页搜索使用原生 `button`，分类使用语义化链接；
- 所有交互元素具有可见的 `:focus-visible` 状态；
- 颜色不作为区分章节的唯一手段，同时保留文字与编号；
- 明暗主题均满足正文和链接的可读对比度；
- 动效在 `prefers-reduced-motion: reduce` 下关闭；
- 保留正确的标题层级和中文页面语义。

## 验证

实现后执行：

1. `pnpm build`，验证 Astro 构建、链接与类型；
2. 分别检查浅色和深色首页；
3. 检查桌面、平板与手机断点；
4. 验证首页搜索按钮和 `Ctrl/⌘ + K`；
5. 键盘遍历全部章节与常用入口；
6. 检查五个章节、常用入口、加入社区和免责声明内容；
7. 确认文档页、侧边栏和 404 页面未受首页样式影响。

## 非目标

- 不替换 Galaxy 主题；
- 不修改文档页视觉；
- 不新增后端、个性化仪表盘或内容推荐算法；
- 不引入校徽、校园照片或未经授权的品牌资产；
- 不调整文档目录和现有路由。
