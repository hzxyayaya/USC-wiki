# 软件推荐

这里收集了同学们推荐的好用的电脑和手机软件，提升你的学习和生活效率。

## 电脑软件 (Windows/Mac)

### 生产力工具
| 软件名称 | 适用平台 | 简介 | 推荐指数 |
| :--- | :--- | :--- | :--- |
| **Notion** | 全平台 | 强大的笔记和知识管理工具，支持数据库功能。 | ⭐⭐⭐⭐⭐ |
| **Obsidian** | 全平台 | 本地优先的双向链接笔记软件，插件生态丰富。 | ⭐⭐⭐⭐⭐ |
| **Snipaste** | Win/Mac | 最好用的截图和贴图工具，没有之一。 | ⭐⭐⭐⭐⭐ |

### 开发工具
- **VS Code**: 宇宙第一编辑器，轻量且强大。
- **GitKraken**: 颜值极高的 Git 图形化客户端。

## 手机 APP (iOS/Android)

- **Forest 专注森林**: 用种树的方式帮你戒掉手机瘾，保持专注。
- **白描**: 准确高效的 OCR 文字识别工具。
