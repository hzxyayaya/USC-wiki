# 计算机学院

欢迎来到计算机学院（CS）的主页。这里汇集了计算机科学与技术、软件工程等专业的课程信息和学习资源。

## 热门课程

### 数据结构与算法
- **课程代码**: CS101
- **学分**: 4.0
- **简介**: 计算机科学的核心课程，涵盖线性表、树、图等数据结构以及排序、查找等算法。
- **资料**: [点击下载复习笔记](#) (示例链接)

### 操作系统
- **课程代码**: CS102
- **学分**: 3.5
- **简介**: 深入理解操作系统的原理，包括进程管理、内存管理、文件系统等。

## 实验室介绍

- **AI 实验室**: 位于科技楼 501，主要研究方向为深度学习和计算机视觉。
- **网络安全实验室**: 位于科技楼 503，专注于网络攻防和信息安全。
