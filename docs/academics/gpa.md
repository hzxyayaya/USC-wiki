# GPA 政策 📈

## 计算公式

学校采用 4.0 学分制。GPA 计算公式如下：

$$ GPA = \frac{\sum (课程学分 \times 课程绩点)}{\sum 课程学分} $$

## 绩点对照表

其中，课程绩点与分数的对应关系通常为：

$$ 绩点 = \begin{cases} 4.0 & \text{if } 分数 \ge 90 \\ 3.0 & \text{if } 80 \le 分数 < 90 \\ 2.0 & \text{if } 70 \le 分数 < 80 \\ 1.0 & \text{if } 60 \le 分数 < 70 \\ 0 & \text{if } 分数 < 60 \end{cases} $$

具体计算方法请参考学生手册[^1]。

[^1]: 学生手册 2025 版，第 23 页，教务处编印。
